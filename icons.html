Usage Guidelines

Consistency

Use icons consistently across the interface. An icon should always represent the same concept or action.

Clarity

Select icons that clearly represent the intended action or concept. When in doubt, use labels alongside icons.

Spacing

Always maintain proper spacing around icons. As a rule of thumb, leave at least 8px of space on all sides.

Color Usage

Use the primary color palette for interactive icons. Non-interactive icons should use neutral colors.
