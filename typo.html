Line Heights

Headings: 1.2

This is an example of heading text with a line height of 1.2. It should be clear and easy to read without too much space between lines.

Body Text: 1.6

This is an example of body text with a line height of 1.6. It provides enough space between lines for comfortable reading, especially in longer paragraphs of text. The extra space helps readers scan text more efficiently.
